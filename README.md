# Complaint Management System

A full-stack platform designed to simplify the process of reporting, tracking, and resolving grievances.

### Key Features
* **User Dashboard:** Submit and track the status of complaints in real-time.
* **Admin Panel:** Manage, update, and resolve issues efficiently.
* **Authentication:** Secure login for users and administrators.

### Tech Stack
* MongoDB, Express.js, React, Node.js (MERN)
* JWT for Authentication
